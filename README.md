## 📁 هيكل الملفات

```
yasser_otba/
├── index.html              ← الصفحة الرئيسية
├── css/
│   └── main.css            ← جميع الأنماط (CSS)
├── js/
│   ├── data.js             ← قاعدة البيانات والمساعدين
│   ├── auth.js             ← نظام المصادقة
│   └── app.js              ← أدوات مشتركة
└── pages/
    ├── auth.html           ← تسجيل الدخول / إنشاء حساب
    ├── dashboard.html      ← لوحة المستخدم
    ├── admin.html          ← لوحة الإدارة
    ├── library.html        ← المكتبة القانونية
    ├── docview.html        ← عرض الوثيقة
    └── subscription.html   ← الاشتراكات والدفع
```

---

## 🚀 طريقة التشغيل المحلي

### الطريقة الأسهل (بدون سيرفر):
افتح `index.html` مباشرة في المتصفح (Chrome أو Firefox أو Edge).

### الطريقة الأفضل (مع سيرفر محلي):
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve .

# ثم افتح: http://localhost:8000
```

## 🌐 رفع المشروع على الإنترنت

### GitHub Pages (مجاناً):
1. أنشئ مستودعاً جديداً على GitHub
2. ارفع جميع ملفات المشروع
3. فعّل GitHub Pages من إعدادات المستودع
4. سيكون رابطك: `https://username.github.io/repo-name`

### Netlify (مجاناً):
1. اذهب إلى netlify.com
2. اسحب مجلد المشروع كاملاً إلى الموقع
3. سيُنشر تلقائياً برابط مجاني

### cPanel (استضافة مدفوعة):
1. اضغط الملفات في zip
2. ارفعهم إلى مجلد `public_html`
3. فك الضغط

## ⚙️ التعديل على المشروع

- **تغيير الألوان:** افتح `css/main.css` وعدّل متغيرات `:root`
- **إضافة مستخدمين:** افتح `js/data.js` وعدّل `DEFAULT_USERS`
- **إضافة وثائق:** سجّل دخول كمدير واستخدم لوحة الإدارة
- **تغيير حد النسخ:** في `js/data.js` عدّل دالة `getLimit()`

---

## 📝 ملاحظات تقنية

- البيانات تُخزَّن في `localStorage` بالمتصفح
- لا يحتاج المشروع قاعدة بيانات خارجية للتشغيل المحلي
- لرفعه على سيرفر حقيقي مع قاعدة بيانات، تحتاج تحويله لـ PHP/Node.js
- المشروع متجاوب بالكامل مع الهواتف والتابلت
